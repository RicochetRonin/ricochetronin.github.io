PPJ 09

What went right:

Team had great leadership and communication/organization during meetings.
Everyone was able to contribute greatly to the project each week.
Everyone on the team was very helpful and willing to give assistance with anything I was having issues with.

What went wrong:

Organization started to slowly deteriorate towards the end of the project.
As a group we lacked artists and had an overabundance of developers which led to some deficiency on the art side of the project.
A lot of time was spent learning some of the tools that we were using, such as CineMachine.

Lessons Learned Going Forward:

Communication is essential to be successful for a project such as this.
Diversify my skill set so I can work more tasks other than just development.

Changes for next project

If I were to work on another game dev project, I would make sure I spend more time learning the tools that we’re going to use in the beginning before production ramps up. I would also make sure I’m better with communication and updating the status of my tasks.
